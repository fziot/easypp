## 背景简介

最初是整合的一个最小版本的设备运营平台, 包含设备运营、扫码支付和前置服务,在业余时间对业务的提炼,之前经历一段时间的整合,但并无支持实际业务
在yq开放后,因某个业务需要拓展市场,和平台需求也相差不大,因此做了一个MVP版本支撑业务,最快的速度测试市场反应,至今……
